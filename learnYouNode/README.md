# 参考资料

祝你好运！

# 项目依赖

整个Workshopper Answers共享依赖，请在`workshopper-answers`根目录下运行以下命令进行安装：

```bash
npm i
```

当然，您也可以使用淘宝镜像加速下载，方法见[这里](https://npm.taobao.org/)。
